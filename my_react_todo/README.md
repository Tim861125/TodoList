# React 待辦事項清單

一個使用 React 和 Vite 建立的簡單待辦事項應用程式。

## 線上展示

[https://tim861125.github.io/TodoList/](https://tim861125.github.io/TodoList/)

## 功能

- 新增待辦事項
- 編輯待辦事項
- 刪除待辦事項
- 標記完成/未完成

## 技術棧

- React 18
- Vite
- react-icons

## 安裝與執行

```bash
# 安裝相依套件
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 部署

```bash
npm run deploy
```

此指令會自動建置並部署至 GitHub Pages。
